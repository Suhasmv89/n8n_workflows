🔮 Predicting Next Customer Orders

This project demonstrates how to use n8n to predict the next order date for customers based on their past purchases. The workflow integrates order history data, applies prediction logic, and notifies customers when they are likely due for their next order.

⚙️ Workflow Overview

The workflow fetches historical orders, applies a simple ML/logic layer to forecast the next likely purchase date, and then triggers a notification via email/SMS if the date is approaching.

✨ Key Features

  • 📊 Fetch and process order history
  • 🤖 Predict next order date (rule-based + optional AI)
  • 📧 Send reminder emails/SMS to customers
  • 📈 Dashboard to monitor predictions and accuracy
